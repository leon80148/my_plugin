# 文章結構模板

## 標題取向（三選一）

### 安心型
- 你擔心的 X，多數時候其實是 Y
- X 讓你焦慮嗎？先別急，看完這篇再說
- 關於 X，你可以先放下的 3 個擔心

### 行動型
- X 發作時先做這 3 步，什麼狀況要快就醫
- 遇到 X 怎麼辦？一張圖看懂處理流程
- X 自救指南：居家照護 vs 該掛急診

### 破解迷思型
- 關於 X 的 5 個常見誤會（第 3 個最多人踩）
- 「X 就是 Y」？醫師告訴你真相
- 別再相信這些 X 偏方了

## 完整 8 段式結構（長文適用）

### 1. 開頭（4-6 句）：共感＋對焦

**公式**：你不是唯一 → 你的擔心合理 → 我們今天把它講清楚 → 你會得到什麼

**句型庫**：
- 「如果你也正在因為 ___ 而焦慮，你不是太敏感，這真的會讓人不安。」
- 「先說結論：大多數 ___ 不代表嚴重問題，但有幾個訊號一定要小心。」
- 「我在門診常被問到這個問題，所以今天整理成一篇，讓你 3 分鐘搞懂。」
- 「你不是第一個這樣擔心的人，事實上這是我被問最多的問題之一。」

### 2. TL;DR（30 秒版本）

用 3-5 點列出：
- 最可能的情況是什麼
- 先做什麼（最小可行動作）
- 什麼情況要就醫（紅旗預告）

**範例**：
```
📌 30 秒重點
• 多數 ___ 是 ___，不是 ___
• 先做：___、___
• 紅旗：出現 ___ 請就醫
• 可以等：症狀 ___ 內改善
```

### 3. 主體：三層資訊

#### 必懂（行動層）
「你要我現在做什麼？」
- 具體步驟，可量化
- 每個建議附適用條件

#### 想懂（原因層）
「為什麼會這樣？」
- 簡化機轉，用比喻
- 解釋「為什麼這樣做有用」

#### 更深入（證據層）
「研究/指引怎麼說？」
- 例外族群
- 專業補充（可摺疊或放文末）

### 4. 自我照護建議

**固定結構**：
```
✅ 適用情境
[輕症/短期/無紅旗的描述]

📋 怎麼做
1. [步驟一，可量化]
2. [步驟二，可量化]
3. [步驟三，可量化]

⏰ 觀察時間
[多久應該改善？例如 48-72 小時]

🛑 停止＋就醫
出現以下情況請停止自我照護，盡快就醫：
- [情況 1]
- [情況 2]
```

### 5. 紅旗區

**標題選項**：
- 「⚠️ 以下情況不要硬撐，請盡快就醫」
- 「🚨 這些訊號出現，請立刻處理」
- 「什麼時候該掛急診？」

**呈現方式**：
- 清單條列，每項一句話
- 可加 emoji 增加辨識度
- 嚴重程度可分級（急診 vs 門診）

### 6. 常見 Q&A / 迷思破解

**格式**：
```
❓ [問題，用口語]
→ [答案，2-4 句，帶原因]
```

**選題原則**：
- 大家會問但不敢問的
- 網路上常見的錯誤觀念
- 門診最常被問的問題

### 7. 結尾：一個可做到的下一步

**句型**：
- 「如果你只想先做一件事：今天先 ___，並且留意 ___。」
- 「你不需要一次做到完美，先把風險排除、把身體照顧好就很棒了。」
- 「記住：___ 是關鍵，其他的可以慢慢來。」

### 8. 免責聲明

**溫柔版（推薦）**：
```
📝 小提醒
這篇是一般衛教整理，不能取代你的主治醫師評估。
若你有慢性病、正在懷孕、或症狀快速惡化，建議直接就醫比較安全。
有任何疑慮，找你信任的醫療團隊聊聊最安心。
```

**正式版**：
```
免責聲明：本文為健康資訊分享，不構成醫療診斷或治療建議。
個人健康問題請諮詢專業醫療人員。
```

## 短文結構（IG/Threads 適用）

```
[Hook：1 句話抓注意力]

📌 重點
• [點 1]
• [點 2]
• [點 3]

⚠️ 紅旗
出現 ___、___、___ 請就醫

💡 今天就做
[一個具體行動]

---
⚕️ 衛教分享，非個人診斷
```

## 影片稿結構

```
[0:00-0:03] Hook
「[問題/痛點/驚訝點]」

[0:03-0:15] 重點預告
「今天告訴你 3 件事：___、___、___」

[0:15-0:45] 主體
[每點 10 秒，用口語]

[0:45-0:55] 紅旗
「但如果你有 ___，不要等，直接就醫」

[0:55-1:00] CTA
「覺得有用按讚收藏，有問題留言」
```
