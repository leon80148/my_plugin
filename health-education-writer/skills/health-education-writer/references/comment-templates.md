# 留言回覆模板

## 三不原則

1. **不診斷**：不在留言區下結論
2. **不開藥**：不給處方級指示
3. **不延誤**：一律先排除紅旗

---

## 回覆模板

### 模板 1：一般症狀詢問

適用：讀者描述症狀，詢問「這是什麼」

```
謝謝你的分享 🙏

因為我沒辦法了解你完整的病史和身體狀況，不太適合在留言直接判斷原因。

不過可以先幫你確認一下：
如果你有 [紅旗症狀 1]、[紅旗症狀 2]，建議盡快就醫/急診。

如果沒有上述情況、症狀已經持續超過 ___ 天，建議安排門診讓醫師評估會比較安心。

這篇文章的「___」段落有整理一些你可以先做的事，可以參考看看！
```

### 模板 2：詢問是否需要就醫

適用：讀者問「我這樣要看醫生嗎」

```
會擔心很正常，我理解 💙

簡單判斷原則：
✅ 可以先觀察：[描述輕症情況]
⚠️ 建議就醫：[描述需就醫情況]
🚨 盡快急診：[描述嚴重情況]

如果你不確定自己屬於哪一種，或是心裡還是很擔心，那就去看一下吧！
讓專業的人幫你確認，安心最重要。
```

### 模板 3：詢問藥物/治療方法

適用：讀者問「可以吃什麼藥」「怎麼治療」

```
這個問題需要醫師看過你的狀況才能判斷喔！

因為每個人的體質、病史、目前用藥都不同，
藥物的選擇和劑量真的要讓醫師評估才安全。

如果你想先了解一般處理原則，這篇文章的「___」段落有整理，
但實際怎麼做還是要跟你的醫師討論 🙏
```

### 模板 4：分享個人經驗，詢問意見

適用：讀者說「我之前也這樣，後來...這樣對嗎」

```
謝謝你分享你的經驗！

每個人的狀況可能不太一樣，你的處理方式適不適合繼續，
建議還是跟你的主治醫師確認一下比較安心。

如果你目前沒有不舒服、狀況穩定，可以在下次回診時問問醫師。
如果有新的症狀或不舒服，建議提早就醫評估喔！
```

### 模板 5：情緒焦慮的讀者

適用：讀者明顯焦慮、反覆詢問、擔心罹患重病

```
我能感受到你現在很擔心，這種不確定的感覺真的很不舒服 💙

網路上的資訊有時候會讓人越查越怕，我建議：

1. 先深呼吸，你願意主動了解已經很棒了
2. 把你擔心的症狀記錄下來
3. 安排一次門診，讓醫師幫你檢查和說明

有專業的人跟你說「沒事」，會比自己猜來猜去安心很多。
照顧好自己，你值得 ❤️
```

### 模板 6：批評/質疑文章內容

適用：讀者質疑資訊正確性或表達不同意見

```
謝謝你的回饋！

你提到的觀點確實值得討論。
這篇文章是根據 [指引/來源] 整理的一般原則，
但醫學確實有很多灰色地帶和例外情況。

如果你有不同的經驗或看法，歡迎分享，
也可以跟你的醫師討論看看你的情況適合怎麼處理。

感謝你願意一起讓這個討論更完整 🙏
```

### 模板 7：詢問特定產品/保健食品

適用：讀者問「XX 產品有效嗎」「可以吃 XX 保健食品嗎」

```
關於特定產品的效果，我沒辦法給你肯定的答案 🙏

因為：
1. 每個人體質和需求不同
2. 產品品質和成分差異大
3. 有些可能跟你的藥物或疾病有交互作用

建議你可以：
• 帶產品成分表問問你的醫師/藥師
• 確認是否有合格認證
• 不要因為吃保健食品就停掉正規治療

有任何疑慮，專業人員的意見最可靠！
```

## 私訊回覆原則

私訊比留言更私密，讀者可能會透露更多細節，但回覆原則相同：

1. **更溫柔**：私訊代表讀者信任你，用更關心的語氣
2. **更謹慎**：越詳細的症狀描述越容易讓人想「診斷」，要克制
3. **更明確導流**：直接建議就醫科別，降低讀者的決策負擔

### 私訊回覆範例

```
收到你的訊息了，謝謝你願意跟我分享 💙

聽起來你最近真的很不舒服，我能理解那種擔心的感覺。

不過因為我沒辦法看到你的檢查結果、也不清楚你完整的狀況，
真的不適合在這裡幫你判斷原因。

我的建議是：
👉 如果你有 [紅旗症狀]，今天就去急診
👉 如果沒有，建議這幾天掛 [建議科別] 門診

去之前可以把你的症狀（什麼時候開始、多嚴重、什麼會加重/減輕）
簡單記錄一下，看診時會更有效率。

祝你早日康復，有問題歡迎再問（但還是要看醫生喔！）🙏
```

## 不適合回覆的情況

以下情況建議不回覆或只給最簡短的導流：

1. **明顯在釣魚/找碴**：不需要浪費能量
2. **已經在其他貼文重複問過**：禮貌請他看之前的回覆
3. **要求私下諮詢/視訊看診**：明確說明你不提供線上診療
4. **涉及法律/醫療糾紛**：建議尋求專業法律協助
5. **推銷產品/詐騙連結**：直接刪除或封鎖
