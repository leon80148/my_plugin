## 理財教育（financial）

### 領域定位
協助讀者建立正確的理財觀念，涵蓋個人理財、投資入門、保險知識、稅務常識等。目標是降低理財焦慮、提高財務素養，而非提供個案投資建議。

### 專屬規則

#### ✅ 必須做到
1. 區分「觀念教育」與「投資建議」，明確告知讀者本文屬於前者
2. 說明投資風險與不確定性，避免讓讀者誤以為有「穩賺」方法
3. 提醒讀者考量自身財務狀況、風險承受度、投資期限
4. 涉及稅務時標註適用地區與年度
5. 數字範例需標註「僅為說明用途，非實際預測」

#### ❌ 絕對禁止
1. 不推薦特定金融商品、個股、基金
2. 不保證投資報酬率（「年化報酬率 XX%」需加風險提醒）
3. 不提供個人化理財規劃
4. 不隱瞞風險或只講好處
5. 不使用「穩賺不賠」「被動收入躺著賺」等誤導用語
6. 不批評讀者過去的理財決策

### 警示訊號（何時該尋求專業協助）

#### ⚠️ 建議諮詢財務顧問
- 負債金額超過月收入的 6 倍
- 即將面臨重大財務決策（購屋、創業、退休規劃）
- 投資虧損影響日常生活品質
- 不了解自己持有的金融商品內容
- 收到不明來源的投資邀約或高報酬承諾

#### ⚠️ 需要立即處理
- 收到法院或金融機構的催繳通知
- 疑似遭遇投資詐騙
- 信用卡循環利息持續累積無法清償
- 被不明人士要求提供帳戶或個資

### 語氣指引
- 理財話題容易引發羞愧感（「都是我不會理財」），要特別注意去羞恥化
- 避免炫耀式語氣（「我靠這招賺了XXX」）
- 強調「每個人的起點不同，重要的是開始」
- 用生活化比喻解釋金融概念

### 免責聲明範本

**溫柔版**：
```
這篇是理財觀念分享，不是投資建議喔！
每個人的財務狀況、風險承受度都不同，實際理財決策建議找合格的財務顧問聊聊。
投資一定有風險，請先了解再行動。
```

**正式版**：
```
免責聲明：本文為理財教育內容，不構成投資建議或財務規劃。
文中提及之金融商品或數據僅為說明用途，不代表推薦或保證績效。
投資有風險，請諮詢合格財務顧問並審慎評估。
```

### 證據分級
- A 級：金管會/央行公告、國際金融組織報告（IMF、世界銀行）、大型金融機構研究報告
- B 級：學術期刊、經同行審查的經濟研究
- C 級：業界分析師觀點、理財專家經驗分享（需標註「經驗性整理」）

## 法律常識（legal）

### 領域定位
協助讀者了解日常生活中常見的法律知識，降低法律焦慮感。目標是讓讀者知道自己的基本權益、常見的法律風險，以及何時需要尋求律師協助。

### 專屬規則

#### ✅ 必須做到
1. 標註法規適用的司法管轄區域（如：台灣、中華民國法律）
2. 說明法律會隨時間修訂，建議讀者確認最新版本
3. 區分「法律常識」與「法律意見」，強調本文屬於前者
4. 涉及具體法條時，附上法條名稱與條號
5. 複雜或涉及重大權益的案件，建議尋求律師協助

#### ❌ 絕對禁止
1. 不提供個案法律意見
2. 不替讀者判斷案件勝敗或可能結果
3. 不鼓勵規避法律或走法律灰色地帶
4. 不簡化複雜的法律爭議為單一結論
5. 不使用「一定告得成」「絕對沒問題」等斷言
6. 不鼓勵私下報復或非法手段解決爭議

### 警示訊號（何時該尋求專業協助）

#### ⚠️ 建議諮詢律師
- 收到法院傳票、起訴書或存證信函
- 涉及刑事案件（被告或被害人）
- 簽署重大合約前（房屋買賣、加盟、合夥）
- 勞資爭議無法自行協商解決
- 智慧財產權受侵害

#### ⚠️ 需要立即處理
- 被逮捕或拘留
- 收到限期回覆的法律文件
- 發現自己可能是詐騙集團人頭帳戶
- 家暴或人身安全受威脅（同時報警）

### 語氣指引
- 法律用語容易讓人畏懼，盡量用白話文解釋
- 避免讓讀者覺得「不懂法律是自己的錯」
- 強調「法律是保護你的工具，不是懲罰你的武器」
- 案例說明時務必去識別化

### 免責聲明範本

**溫柔版**：
```
這篇是法律常識整理，不是針對你個人狀況的法律意見喔！
法律因地區、時間、個案情況都會不同，如果你遇到具體問題，建議找律師聊聊最安心。
很多縣市都有免費法律諮詢資源，可以善加利用。
```

**正式版**：
```
免責聲明：本文為法律常識分享，不構成法律意見。
法律隨時間修訂，文中資訊以撰文時適用之法規為準。
個案問題請諮詢專業律師。
```

### 證據分級
- A 級：現行法律條文、司法院解釋、最高法院判例
- B 級：高等法院判決、法學期刊論文、律師公會出版品
- C 級：律師實務經驗、法律評論（需標註「實務觀察」）

## 營養知識（nutrition）

### 領域定位
提供正確的飲食觀念與營養素知識，破解常見飲食迷思。目標是讓讀者建立健康的飲食認知，而非提供個人化飲食處方。

### 專屬規則

#### ✅ 必須做到
1. 區分「一般飲食建議」與「醫療營養治療」
2. 說明個體差異（年齡、性別、活動量、健康狀況會影響需求）
3. 標註資訊來源（國民飲食指南、營養學會建議等）
4. 涉及特殊飲食法時，說明適用對象與潛在風險
5. 保健食品相關內容需提醒「食品非藥品」

#### ❌ 絕對禁止
1. 不開立個人化飲食處方
2. 不推薦特定品牌的保健食品
3. 不使用「排毒」「燃脂」等缺乏科學根據的用語
4. 不將單一食物神化或妖魔化
5. 不提供極端飲食法（極低卡路里、單一食物飲食等）
6. 不批評讀者的體型或飲食習慣

### 警示訊號（何時該尋求專業協助）

#### ⚠️ 建議諮詢營養師/醫師
- 有特殊疾病需要飲食控制（糖尿病、腎臟病、食物過敏）
- 計畫進行大幅度飲食改變
- 長期體重異常變化（非刻意減重卻持續變輕）
- 懷孕或哺乳期的營養需求
- 兒童或青少年的發育營養問題

#### ⚠️ 需要立即處理
- 食物過敏反應（呼吸困難、全身蕁麻疹）→ 同時屬於醫療急診
- 疑似食物中毒（劇烈嘔吐、腹瀉、發燒）
- 進食障礙相關行為（暴食後催吐、極端限制飲食）

### 語氣指引
- 飲食是高度私人化的行為，避免批判
- 不要製造飲食焦慮（「你吃的都是毒」）
- 強調「沒有完美的飲食，重要的是整體模式」
- 用「可以試試看」取代「你必須這樣吃」

### 免責聲明範本

**溫柔版**：
```
這篇是一般飲食觀念分享，每個人的身體狀況和營養需求都不同。
如果你有特殊健康狀況或飲食限制，建議找營養師或醫師討論最適合你的方案。
健康飲食沒有標準答案，找到適合自己的節奏最重要。
```

**正式版**：
```
免責聲明：本文為營養知識分享，不構成個人化飲食建議。
特殊健康狀況者請諮詢專業營養師或醫師。
```

### 證據分級
- A 級：國民飲食指南、WHO 營養建議、系統性回顧
- B 級：大型營養流行病學研究、隨機對照試驗
- C 級：專家共識、營養師臨床觀察（需標註「經驗性整理」）

## 科技安全（tech-safety）

### 領域定位
協助讀者了解數位時代的資訊安全與隱私保護知識，包含防詐騙、密碼安全、個資保護、裝置安全等。目標是降低科技焦慮、提高數位素養。

### 專屬規則

#### ✅ 必須做到
1. 提供具體、可操作的防護步驟
2. 說明威脅的嚴重程度與發生機率，避免過度恐慌
3. 區分「一般用戶」與「高風險用戶」的防護需求
4. 涉及軟體/工具時，說明官方取得管道
5. 資安事件案例需去識別化

#### ❌ 絕對禁止
1. 不教導攻擊手法的具體操作細節
2. 不推薦未經驗證的資安工具
3. 不恐嚇式行銷（「你的手機隨時被監聽」）
4. 不洩漏尚未公開修補的漏洞細節
5. 不鼓勵讀者自行「反擊」或「追蹤」攻擊者

### 警示訊號（何時該尋求專業協助）

#### ⚠️ 建議尋求專業協助
- 帳號被盜且無法自行恢復
- 發現裝置有不明程式或異常行為
- 企業或組織遭受資安攻擊
- 個資已外洩且收到異常通知

#### ⚠️ 需要立即處理
- 遭遇網路詐騙且已匯款 → 立即撥打 165 反詐騙專線
- 收到勒索軟體通知
- 發現銀行帳戶有不明交易
- 遭受網路霸凌或威脅 → 保留證據並報警

### 語氣指引
- 科技話題容易讓非技術背景的讀者感到自卑，要特別耐心
- 避免使用過多技術術語，用生活化比喻
- 不要嘲笑「低級錯誤」（點到釣魚連結不是笨）
- 強調「資安是習慣，不是一次性設定」

### 免責聲明範本

**溫柔版**：
```
這篇是資安觀念分享，科技環境變化很快，建議定期更新你的防護措施。
如果你遇到具體的資安事件，可以撥打 165 反詐騙專線或尋求專業資安人員協助。
做好基本防護就已經很棒了，不需要成為資安專家。
```

**正式版**：
```
免責聲明：本文為資訊安全教育內容，不保證能防範所有網路威脅。
遇到資安事件請聯繫專業人員或相關主管機關。
```

### 證據分級
- A 級：政府資安主管機關公告（TWCERT/CC）、國際資安組織報告（NIST、OWASP）
- B 級：資安廠商研究報告、學術論文
- C 級：資安社群分析、業界經驗（需標註「實務觀察」）

## 育兒教養（parenting）

### 領域定位
提供以實證為基礎的育兒知識與教養觀念，涵蓋兒童發展、親子溝通、行為引導等。目標是降低育兒焦慮、增加父母信心，而非判斷「好父母」或「壞父母」。

### 專屬規則

#### ✅ 必須做到
1. 說明兒童發展有個體差異，避免讓家長過度比較
2. 區分「發展參考範圍」與「診斷標準」
3. 涉及行為問題時，提供正向引導策略
4. 標註建議的適用年齡範圍
5. 涉及兒童健康問題時，導流至醫療領域

#### ❌ 絕對禁止
1. 不判斷家長是否稱職
2. 不推廣體罰或威脅式管教
3. 不比較不同家庭/孩子的發展速度
4. 不將單一教養風格塑造為唯一正解
5. 不提供兒童心理疾病的診斷
6. 不批評不同文化的育兒方式

### 警示訊號（何時該尋求專業協助）

#### ⚠️ 建議諮詢兒童發展專家
- 孩子的發展明顯落後同齡平均（語言、動作、社交）
- 行為問題持續影響日常生活和學習
- 親子關係長期緊張，互動模式固化
- 孩子出現退化行為（已經會的能力突然不會）

#### ⚠️ 需要立即處理
- 孩子表達自傷或傷害他人的意圖
- 發現孩子遭受霸凌、虐待或性侵
- 家長自身情緒已到失控邊緣 → 先確保安全，再尋求協助
- 孩子出現嚴重飲食或睡眠問題

### 語氣指引
- 育兒議題極容易引發罪惡感，去羞恥化特別重要
- 避免「別人的孩子都可以，你的孩子怎麼不行」的暗示
- 強調「你已經是夠好的父母了」
- 用「你可以試試看」取代「你應該這樣做」
- 承認育兒沒有標準答案，每個家庭都在摸索

### 免責聲明範本

**溫柔版**：
```
這篇是育兒觀念分享，每個孩子和家庭都是獨特的。
如果你對孩子的發展有疑慮，建議找兒科醫師或兒童發展專家聊聊。
你願意為孩子查資料、想變得更好，這本身就是很棒的事。
```

**正式版**：
```
免責聲明：本文為教養觀念分享，不構成兒童發展評估或心理諮商。
兒童發展問題請諮詢專業醫療或教育人員。
```

### 證據分級
- A 級：國際兒科學會指引、WHO 兒童發展標準、系統性回顧
- B 級：發展心理學研究、大型縱貫研究
- C 級：教育現場觀察、資深教養專家經驗（需標註「經驗性整理」）

## 自訂領域（custom）

### 使用方式

當使用者指定 `domain: custom` 時，需要額外提供以下資訊：

| 欄位 | 說明 | 是否必填 |
|------|------|----------|
| domain_name | 自訂領域名稱 | 必填 |
| professional_title | 該領域的專業人士稱呼 | 必填 |
| must_do | 該領域必須做到的規則 | 建議填寫 |
| must_not | 該領域絕對禁止的規則 | 建議填寫 |
| warning_signals | 何時該尋求專業協助 | 建議填寫 |
| disclaimer | 免責聲明 | 建議填寫 |

### 最小設定範例

```
domain: custom
domain_name: 寵物照護
professional_title: 獸醫師
```

系統會自動套用通用規則，並將「專業人士」替換為指定的稱呼。

### 完整設定範例

```
domain: custom
domain_name: 寵物照護
professional_title: 獸醫師
must_do:
  - 區分一般照護建議與獸醫診療
  - 標註適用的動物種類
  - 不同體型/品種的注意事項
must_not:
  - 不提供處方藥劑量
  - 不替代獸醫診斷
warning_signals:
  - 寵物拒食超過 24 小時
  - 呼吸困難或異常
  - 大量出血或嚴重外傷
disclaimer: 本文為寵物照護知識分享，不構成獸醫診斷或治療建議。寵物健康問題請諮詢專業獸醫師。
```

## 領域切換注意事項

### 跨領域內容
部分主題可能跨越多個領域（例如：「糖尿病飲食」同時涉及 medical 與 nutrition）。處理方式：

1. 以主要領域為主，套用該領域的完整規則
2. 涉及次要領域時，額外標註該領域的免責聲明
3. 在文末同時列出兩個領域的專業諮詢建議

### 領域預設排除範圍

| 領域 | 預設 scope_limit |
|------|-----------------|
| medical | 孕婦/幼兒/特殊疾病另行就醫 |
| financial | 法人/企業財務/跨國稅務另行諮詢 |
| legal | 跨國法律/刑事重案另行諮詢律師 |
| nutrition | 重大疾病飲食控制另行諮詢營養師 |
| tech-safety | 企業級資安/國家級攻擊另行處理 |
| parenting | 特殊需求兒童另行諮詢專業團隊 |
