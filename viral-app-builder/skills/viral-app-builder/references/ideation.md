# 創意發想與市場驗證

## 腦暴創新點子

### 鏈式提問法
1. **廣泛發想**：列出10個領域問題與解決方案
2. **聚焦細節**：選出最有潛力的點子展開 UVP、MVP功能、目標用戶、技術需求
3. **橫向擴展**：探索跨行業應用可能性

### 創意約束激發法
添加限制條件來激發更具體的創意：
- 「30天內可盈利、啟動資金低於1000美元、可由1人經營」
- 「目標客戶是中小企業」
- 「結合 {熱門科技} 與 {傳統行業}」

### 提示範本

**廣泛發想：**
```
請列出10個 {你關注的領域} 中新近冒出的問題與對應解決方案，
並標註目標客群。格式：問題 : 解決方案 : 目標用戶
```

**深入展開：**
```
針對「{選中的點子}」，請展開以下細節：
1. 獨特價值主張 (UVP) 是什麼？
2. MVP 需要哪些核心功能？
3. 潛在早期用戶是誰？
4. 實現需要的主要技術條件？
```

**橫向擴展：**
```
如何將「{解決方案}」應用到完全不同的行業？
請舉5個出人意料的用例。
```

## 市場驗證

### 避免 AI 幻覺
- 批判性思維看待 AI 建議
- 輔以人工查證市場數據
- 確認競品、市場規模、用戶痛點真實存在

### 競品分析提示
```
請列出「{你的點子}」的現有競爭對手或替代方案，
分析我的想法與它們的差異。
```

### 用戶研究準備
```
請幫我起草一份用戶調研問卷，
目的是驗證「{產品概念}」的市場需求。
包含：篩選問題、痛點探索、付費意願、功能偏好。
```

## 驗證清單

- [ ] 確認目標市場規模足夠大
- [ ] 確認用戶痛點真實存在且足夠強烈
- [ ] 了解現有競品的優劣勢
- [ ] 明確你的獨特價值主張
- [ ] 有初步的用戶反饋支持你的假設
