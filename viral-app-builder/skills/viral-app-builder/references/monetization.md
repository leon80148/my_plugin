# 營利模式與商業化策略

## 常見營利模式概覽

| 模式 | 說明 | 優點 | 缺點 | 適用場景 |
|------|------|------|------|----------|
| 付費下載 | 一次性購買 | 簡單直接 | 門檻高、後續收入有限 | 專業工具 |
| 免費+廣告 | 廣告變現 | 覆蓋面廣 | 影響體驗 | 內容/工具類 |
| Freemium | 基礎免費、高階付費 | 兼顧用戶量和收益 | 需精心設計分界 | 大多數 App |
| 訂閱制 | 週期性付費 | 穩定收入 | 需持續提供價值 | SaaS、內容平台 |
| 內購 | 購買虛擬商品 | 深度用戶高付費 | 可能影響平衡 | 遊戲、社交 |

### 模式評估請求
```
對於一款 {產品類型} 應用，哪種營利模式比較適合？

請比較：
1. {模式A} vs {模式B}
2. 各自的優劣勢
3. 推薦方案及理由
```

## Freemium 設計

### 免費 vs 付費功能劃分原則
- 免費版應能體驗核心價值
- 付費版提供進階/便利功能
- 避免「閹割」感（免費版要有用）
- 自然引導升級（不強迫）

### 常見付費功能
- 去除廣告
- 雲端同步
- 進階統計/分析
- 更多自定義選項
- 優先客服支援
- 匯出功能

### 設計請求
```
我的 {產品類型} 採用 Freemium 模式，
請建議：
1. 免費版應包含哪些功能？
2. 付費版獨有哪些功能？
3. 如何自然引導用戶升級？
```

## 訂閱制設計

### 定價層級設計
```
請為 {產品類型} 設計訂閱方案：
1. 基礎方案（輕度用戶）
2. 專業方案（重度用戶）
3. 團隊/家庭方案（可選）

包含：價格建議、功能差異、付費週期。
```

### 訂閱定價考量
- 競品價格參考
- 用戶感知價值
- 月付 vs 年付折扣（通常年付打8折）
- 免費試用期長度（7天 vs 14天）

## 廣告營收優化

### 廣告類型
| 類型 | 說明 | 體驗影響 | 收益 |
|------|------|----------|------|
| 橫幅廣告 | 固定位置小幅廣告 | 低 | 低 |
| 插頁廣告 | 全螢幕切換時顯示 | 中 | 中 |
| 獎勵視頻 | 看廣告換獎勵 | 低（主動選擇） | 高 |
| 原生廣告 | 融入內容的廣告 | 低 | 中 |

### 廣告體驗平衡
```
在哪些使用情境下插入廣告比較合適，不容易讓用戶反感？

產品類型：{產品描述}
用戶行為模式：{使用情境}
```

**最佳實踐：**
- 在功能流程的自然間歇展示
- 完成任務後而非進行中
- 控制頻率（如每5分鐘最多1次）
- 提供付費去廣告選項

## 個人化定價與 AI 分析

### 差異化定價設計
```
請根據不同用戶群的使用模式，
設計區隔的訂閱方案：
- 輕度用戶
- 重度用戶  
- 團隊用戶

用戶數據：{使用模式描述}
```

### 智慧推薦時機
- 偵測高付費潛力用戶
- 適當時機推送升級優惠
- 避免打擾低意願用戶

## 法規合規與道德考量

### 合規檢查清單
- [ ] GDPR 合規（歐盟用戶）
- [ ] CCPA 合規（加州用戶）
- [ ] COPPA 合規（如有兒童用戶）
- [ ] 隱私政策完整且易讀
- [ ] 訂閱條款清楚告知
- [ ] 取消訂閱流程簡單

### 避免的做法
- 套路訂閱（試用後自動扣款不提醒）
- 隱藏費用
- 取消流程過於複雜
- 過度收集用戶數據
